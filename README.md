# 🖼️ AI Image Editing & Background Replacement Tool

## 🔍 Overview
An interactive web-based image editing application that enables background replacement and image manipulation using computer vision techniques.

Built with Python, OpenCV, and Streamlit, this project demonstrates practical implementation of image processing in a user-friendly interface.

---

## 🎯 Problem Statement
Manual image editing tools can be:
- Time-consuming  
- Skill-intensive  
- Not easily accessible for quick edits  

This project simplifies image editing by providing an intuitive interface powered by automated computer vision techniques.

---

## ⚙️ Key Features
- 🖼️ Background removal and replacement  
- 🎨 Image editing using OpenCV operations  
- ⚡ Real-time processing via Streamlit interface  
- 📤 Upload and process custom images  
- 🧠 Automated image manipulation workflows  

---

## 🧠 Core Technologies
- **Python** – Core logic  
- **OpenCV** – Image processing and transformation  
- **Streamlit** – Interactive web interface

## 🏗️ System Architecture
- **Frontend** – Streamlit UI (user interaction & display)  
- **Processing Layer** – OpenCV algorithms for image transformation

## ⚡ Installation

```bash
git clone https://github.com/HiyaJain1005/Image_Editing_Site.git
cd Image_Editing_Site
```

Install dependencies:
```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

Then open:
```
http://localhost:8501
```
